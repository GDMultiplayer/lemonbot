# Lemon [discord.js]

Lemon is a Discord moderation, music and fun open-source bot.

## Features:

* Many moderation commands, such as `ban`, `kick`, `slowmode`
* Very fast audio engine node-opus
* High bot respond time

## This bot uses:

* discord.js
* discord.js-commando
* node-mongodb
And many other libs
